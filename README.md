# CGV Demo

Application clone from CGV on appstore

<img src="Screenshots/screenshoot_1.png" width="563" height="1218">

## Requirements

- Runs on iOS 10.0 and later
- Swift 5.0  and ARC.

## Installation

Please run pod install before run project.

```ruby
pod install
```
## Information

- Architecture: Clean Architecture, MVVM.
- Framework: RxSwift
